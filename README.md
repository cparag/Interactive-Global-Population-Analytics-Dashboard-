# Interactive-Global-Population-Analytics-Dashboard-

Built interactive Tableau dashboards to analyze population metrics (life expectancy, birth rate, death rate, fertility, and infant mortality) for 200+ countries.​

Compared top and bottom performing countries by life expectancy and population growth rates, highlighting urban/rural distributions and demographic differences.​

Visualized global and regional patterns in birth and death rates, demonstrating high birth rates in African nations, higher death rates in Europe, and policy-relevant trends in aging and population decline.​

Created heat maps and bar/donut charts to present trends and anomalies, enabling users to track annual changes and compare across regions and years.​

Derived actionable insights for demographic studies, policy planning, and economic forecasting using time-series and comparative analytics.​

Automated dashboard interactivity for end-users, enabling real-time exploration of global population trends by country, year, and metric.​
